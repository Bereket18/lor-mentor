# Lor Mentor

> Next.js 15 web + NestJS 11 API, Prisma 7/PostgreSQL, Redis/BullMQ queues, Google Gemini
> Addis Ababa, Ethiopia.

## What is Lor Mentor?

Lor Mentor transforms every PDF into AI flashcards, quizzes,
summaries, and 3D anatomy experiences — built exclusively
for Lorcan Medical College students.

## Tech Stack

- **Frontend:** Next.js 15, TypeScript, TailwindCSS, ShadCN UI
- **Backend:** NestJS, PostgreSQL, Prisma
- **AI:** Google Gemini API
- **3D:** Three.js + React Three Fiber
- **Deploy:** Hostinger VPS
